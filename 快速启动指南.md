# 快速启动指南

## 🎯 立即体验（3种方式）

### 方式1：演示模式（最快，无需安装）

1. 双击打开 `demo.html` 文件
2. 立即体验基本功能
3. 适合快速预览界面和交互

**功能限制**：无数据持久化、无用户系统

---

### 方式2：完整版本（推荐）

#### 第一步：安装Node.js
- 如果已安装Node.js，跳过此步
- 下载地址：https://nodejs.org/
- 推荐版本：v16+ 

#### 第二步：安装依赖
```bash
cd stock-trading-app
npm install
```
等待安装完成（约1-2分钟）

#### 第三步：启动服务器
```bash
npm start
```

#### 第四步：打开浏览器
访问：`http://localhost:3001`

**如果端口被占用**，服务器会自动选择其他端口，请查看控制台输出的地址。

---

### 方式3：一键启动（智能端口）

```bash
node start.js
```

此脚本会：
1. 自动查找可用端口
2. 修改配置文件
3. 启动服务器
4. 自动打开浏览器

---

## 🎮 使用流程

### 1. 注册账号
- 点击右上角"登录"
- 切换到"注册"标签页
- 填写用户名、密码
- 点击"注册"

### 2. 登录系统
- 使用注册的账号登录
- 系统跳转到大盘页面

### 3. 查看行情
- 浏览三大指数（上证、深证、创业板）
- 查看热门股票价格
- 观察实时价格更新（每3秒）

### 4. 买入股票
- 点击股票行的"交易"按钮
- 输入买入数量（1手 = 100股）
- 点击"确认买入"
- 查看余额变化

### 5. 查看持仓
- 点击左侧"我的持仓"
- 查看所有持仓股票
- 查看盈亏情况

### 6. 卖出股票
- 在持仓页面选择股票
- 输入卖出数量
- 点击"确认卖出"

## 🔧 遇到问题？

### 问题1：端口被占用
**解决**：修改 `backend/server.js` 中的 `PORT` 常量

### 问题2：无法安装依赖
**解决**：
- 检查Node.js是否安装：`node -v`
- 尝试使用国内镜像：`npm install --registry=https://registry.npmmirror.com`

### 问题3：数据库错误
**解决**：删除 `database` 目录，重新启动服务器

### 问题4：前端无法连接后端
**解决**：
- 检查后端是否启动
- 查看浏览器控制台错误
- 确认API地址是否正确

## 📊 数据库初始化

首次运行时会自动创建以下表：

### users表
```sql
CREATE TABLE users (
  id INTEGER PRIMARY KEY,
  username TEXT UNIQUE,
  password TEXT,
  email TEXT UNIQUE,
  balance REAL DEFAULT 100000.00,
  created_at DATETIME
);
```

### holdings表
```sql
CREATE TABLE holdings (
  id INTEGER PRIMARY KEY,
  user_id INTEGER,
  stock_code TEXT,
  stock_name TEXT,
  quantity INTEGER,
  avg_price REAL,
  FOREIGN KEY (user_id) REFERENCES users(id)
);
```

### transactions表
```sql
CREATE TABLE transactions (
  id INTEGER PRIMARY KEY,
  user_id INTEGER,
  stock_code TEXT,
  stock_name TEXT,
  type TEXT,
  quantity INTEGER,
  price REAL,
  total_amount REAL,
  created_at DATETIME,
  FOREIGN KEY (user_id) REFERENCES users(id)
);
```

## 🐛 常见问题

### Q1: 端口被占用
**A**: 修改 `backend/server.js` 中的 `PORT` 常量，或删除 `start.js` 自动查找可用端口。

### Q2: 数据库无法创建
**A**: 确保 `database` 目录存在且有写入权限。当前使用内存模式（`:memory:`），重启后数据会丢失。如需持久化，请修改为文件路径。

### Q3: WebSocket连接失败
**A**: 检查前端 `app.js` 中的Socket.IO地址是否与后端一致。

### Q4: 价格不更新
**A**: 检查浏览器控制台是否有错误，确认WebSocket连接成功。

# 股票模拟交易软件

一个功能完整的股票模拟交易Web应用，类似支付宝的模拟炒股功能。

## 功能特性

### ✅ 已实现功能

1. **用户认证系统**
   - 用户注册（用户名、密码、邮箱）
   - 用户登录（JWT token认证）
   - 密码加密存储（bcryptjs）
   - 登录状态持久化（localStorage）

2. **实时股票数据**
   - 模拟实时股票价格更新（每3秒）
   - 上证指数、深证成指、创业板指
   - 热门股票：贵州茅台、宁德时代、中国平安等
   - WebSocket实时推送数据更新

3. **市场行情页面**
   - 三大指数卡片展示
   - 股票列表（名称、价格、涨跌幅）
   - 股票搜索功能
   - 点击查看股票详情

4. **股票详情页面**
   - 实时价格显示
   - 价格走势图（Canvas绘制）
   - 股票基本信息（成交量、成交额、总市值、市盈率）
   - 买入/卖出按钮

5. **模拟交易功能**
   - 买入股票（支持手数输入）
   - 卖出股票（检查持仓）
   - 实时余额计算
   - 交易确认弹窗

6. **持仓管理**
   - 查看当前持仓
   - 显示成本价、当前价、盈亏
   - 持仓数量统计

7. **交易记录**
   - 完整的买卖记录
   - 交易时间、价格、数量
   - 按时间倒序排列

8. **自选股功能**
   - 添加/删除自选股
   - 自选股列表展示

## 技术栈

### 后端
- **Node.js** + **Express** - Web服务器
- **SQLite3** - 数据库（轻量级，无需安装）
- **Socket.IO** - WebSocket实时通信
- **JWT** - 用户认证
- **bcryptjs** - 密码加密

### 前端
- **原生HTML/CSS/JavaScript** - 无框架依赖
- **Canvas API** - 绘制K线图
- **Socket.IO Client** - 实时数据接收
- **Fetch API** - 与后端通信

## 使用说明

### 首次使用

1. 点击"注册"按钮创建账户
2. 填写用户名、密码、邮箱（可选）
3. 注册成功后自动跳转到登录页面
4. 使用注册的账号登录

### 模拟交易

1. **查看行情**：在"市场行情"页面查看各股票实时价格
2. **搜索股票**：在顶部搜索框输入股票代码或名称
3. **查看详情**：点击股票行查看详细信息
4. **买入股票**：
   - 在股票详情页点击"买入"按钮
   - 输入购买数量（手）
   - 确认交易
5. **卖出股票**：
   - 在"我的持仓"页面查看持仓
   - 点击"卖出"按钮
   - 输入卖出数量
   - 确认交易

### 数据说明

- **初始余额**：¥100,000.00
- **股票价格**：模拟数据，每3秒随机波动±1%
- **交易规则**：
  - 1手 = 100股
  - 买入需要足够余额
  - 卖出需要足够持仓

## 数据库结构

### users表
- id: 用户ID
- username: 用户名（唯一）
- password: 密码（加密）
- email: 邮箱（唯一，可选）
- balance: 账户余额
- created_at: 创建时间

### holdings表
- id: 持仓ID
- user_id: 用户ID
- stock_code: 股票代码
- stock_name: 股票名称
- quantity: 持仓数量
- avg_price: 平均成本价

### transactions表
- id: 交易ID
- user_id: 用户ID
- stock_code: 股票代码
- stock_name: 股票名称
- type: 交易类型（buy/sell）
- quantity: 交易数量
- price: 交易价格
- total_amount: 交易总额
- created_at: 交易时间

### watchlist表
- id: 自选ID
- user_id: 用户ID
- stock_code: 股票代码
- stock_name: 股票名称

## API接口

### 认证相关
- `POST /api/register` - 用户注册
- `POST /api/login` - 用户登录
- `GET /api/user` - 获取用户信息（需认证）

### 股票相关
- `GET /api/stocks` - 获取所有股票数据
- `GET /api/stocks/:code` - 获取特定股票数据

### 交易相关
- `GET /api/holdings` - 获取持仓（需认证）
- `GET /api/watchlist` - 获取自选股（需认证）
- `POST /api/watchlist` - 添加自选股（需认证）
- `DELETE /api/watchlist/:code` - 删除自选股（需认证）
- `POST /api/trade/buy` - 买入股票（需认证）
- `POST /api/trade/sell` - 卖出股票（需认证）
- `GET /api/transactions` - 获取交易记录（需认证）
